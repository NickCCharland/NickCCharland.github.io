Study Spot: A UX Research and Design Project

About This Project

This project was completed in SMAD 317: User Interaction Design. The purpose for creating this project was to explore the fundamentals of User Interaction Design through utilizing the design thinking process. This included empathizing, defining, idealizing, protoyping, and testing an idea for an application that solved a specific issue.
