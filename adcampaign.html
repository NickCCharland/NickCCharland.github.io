Non-profit Advertising Campaign

This project was completed as a final project for SMAD 201: Foundations of Visual Communication Design in December of 2019. The goal of this project was to create an integrated campaign that seeks to motivate the audience to engage in a cause or take action. By using both image and typography, you must consider the overall message.

The campaign had to consist of several deliverables: an 11" x 17" poster, a brochure, and a one-page informational flyer containing an infographic designed for print.



Project Goals

The main idea behind this advertising campaign was to recruit volunteers for Harrisonburg Downtown Renaissance.
